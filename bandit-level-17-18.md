# Bandit Level 17 → 18

## Objectif
Trouver la ligne différente entre deux fichiers de mots de passe.

## Commandes utilisées
diff passwords.old passwords.new

## Ce que j'ai appris
- diff compare deux fichiers ligne par ligne
- < indique la ligne dans le premier fichier
- > indique la ligne dans le deuxième fichier
- Le nouveau mot de passe est la ligne avec >
