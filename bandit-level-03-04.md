# Bandit Level 3 → 4

## Objectif
Trouver un fichier caché dans un dossier.

## Commandes utilisées
ls
cd inhere
ls -la
cat ...Hiding-From-You

## Ce que j'ai appris
- Les fichiers cachés commencent par un point "."
- ls normal ne les affiche pas
- ls -la affiche tous les fichiers y compris les cachés
- Un fichier peut avoir plusieurs points au début du nom
